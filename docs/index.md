# Harmonia Occulta Documentation

Welcome to the documentation for the Harmonia Occulta project. This project focuses on the computational analysis of Robert Fludd's Monochordum Mundi.

## Project Overview

This repository contains code and data for analyzing the harmonic structure of Robert Fludd's 17th-century cosmological diagrams, with a focus on the Monochordum Mundi.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/AlkaiDynamics/Harmonia-Occulta.git
   cd Harmonia-Occulta
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Explore the Jupyter notebooks in the `notebooks/` directory to see the analysis:
- `omr_pipeline.ipynb`: Optical Music Recognition pipeline
- `omr_analysis.ipynb`: Analysis of the monochord structure
